# change-ac-name-bot
The robot changes the name of the account to the watch with Telethon library
## Installation library
1 : Upgarde pip :
```bash
python3 -m pip install --upgrade pip
```
2 : installation :
```bash
python3 -m pip install --upgrade telethon
```
